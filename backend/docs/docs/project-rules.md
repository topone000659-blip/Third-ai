# Third AI Project Rules

## General

- Keep the project modular.
- Build one feature at a time.
- Do not modify generated files manually.
- Test every completed feature before moving on.
- Keep documentation updated.

## Development

- Frontend and backend remain separate.
- AI engine remains independent.
- Use clear folder structures.
- Write maintainable code.

## Quality

- Professional UI/UX.
- High performance.
- Secure architecture.
- Scalable design.
